Prerequisites
- root access to your phone
- a version of su that permits an application to gain root access
- for openvpn to function you need the tun capability in your kernel
- for openvpn to be able to configure the interface you need busybox ifconfig in /system/xbin/bb. Androids ifconfig is not compatible with openvpn.

Version
OpenVPN Installer will install OpenVPN 2.1_rc15
It is statically linked against
- liblzo
- openssl, extended with the blowfish cypher

Links
View the source: openvpn, openssl, liblzo.

Who am I
I'm a software developer, based in Germany. In January 2009 I got a ADP1. I'm a long term OpenVPN user. At the time I could only find a openvpn 2.0 binary for the ADP1 (without source). Around March 2009 I wrote the necessary Android.mk makefiles, made changes to the android supplied openssl and started to work on a management app (OpenVPN-Monitor). Mid September 2009 I put the sources on github. A few days later cyanogen integrated the binary in his famous CyanogenMOD. Credits
